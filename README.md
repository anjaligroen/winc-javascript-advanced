# JavaScript Recipe Management System

This is a command-line recipe management application built with Node.js. It allows users to:

- View unique authors of cake recipes
- List recipe names by author or ingredient
- Search for recipes by name
- View all ingredients of saved recipes
- Save favorite recipes to a personal list

The program reads recipe data from a JSON file (`cake-recipes.json`) and provides an interactive menu for easy navigation.

## Features

- Fetch unique authors from recipes
- Filter recipes by author or ingredient
- Retrieve specific recipes by name
- Manage a saved recipes list with ingredient aggregation
